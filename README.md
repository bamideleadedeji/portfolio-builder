# Financial Portfolio Builder

A professional web application built with Python and Streamlit that allows users to analyze, optimize, and visualize investment portfolios in real-time.

## Live Demo
[https://portfolio-builder-mxqcq9mf5vpwagpb8xdhy3.streamlit.app/]

## Features
* **Real-time Data:** Fetches latest market data using the Yahoo Finance API.
* **Interactive Visuals:** Dynamic charts powered by Plotly for asset allocation and performance history.
* **Portfolio Optimization:** Calculates risk/return metrics to help users make informed decisions.
* **User-Friendly Interface:** Easy-to-use sliders and inputs for custom investment amounts.

## Installation & Local Usage
If you want to run this project locally on your machine:

1. Clone the repository:
   ```bash
   git clone [https://github.com/YOUR_USERNAME/portfolio-builder.git](https://github.com/bamideleadedeji/portfolio-builder.git)

2   Install dependencies:
    pip install -r requirements.txt

3   Run the app:
    streamlit run app.py

    Tech Stack
Language: Python

Framework: Streamlit

Data Analysis: Pandas, NumPy

Visualization: Plotly, Matplotlib
